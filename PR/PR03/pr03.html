ПР03 Использование CSS Flexbox/Grid в верстке веб-страниц

Ваша задача - сверстать главную страницу сайта, в соответствии с прототипом представленном ниже. Для расположения элементов на странице использовать Flexible Box Layout Module и CSS Grid Layout Module.

100% сходство НЕОБЯЗАТЕЛЬНО!! Главное чтобы дизайн страницы соответствовала прототипу, а также основные блоки были на своих местах.

НЕОБЯЗАТЕЛЬНО!! Выбор шрифтов, изображений, наполнение блоков контентом остаётся за студентом.

Для выполнения данной работы разрешается использовать только HTML и CSS. Использование JS или различных библиотек и фреймворков ЗАПРЕЩЕНО; только HTML5/CSS3

Критерии оценивания

  • используются Grid Layout для расположения основных блоков на странице
  • используется Flexbox для позиционирования элементов внутри блоков контейнеров
  • дизайн страницы соответствует прототипу, а также основные блоки находились на своих местах
  • наличе всех блоков на странице(см. макет выше)
  • отсутствие горизонтальных скроллов при любом разрешении экрана(до 480px)
  • качество реализации процесса изменения сайта от одной контрольной точки до другой
  • видоизменение каркаса сайта на каждой контрольной точки

В качестве контрольных точек берутся следующие диапазоны ширины экрана

  • больше или равно 1601px
  • больше 1367px но меньше 1600px
  • больше 1025px но меньше 1366px
  • больше 801px но меньше 1024px
  • больше 481px но меньше 800px
  • менее 480px

Пример выполнения работы

Наполнение сайта контентом необязательно. Допускается разделить страницу на основные блоки("подкрасив их").
