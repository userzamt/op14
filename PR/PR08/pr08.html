ПР08 Работа с классами в JavaScript

Задание

Основываясь на Ваш вариант задания, создать класс "предок". Данный класс должен обладать общими свойствами и методами, которые присущи всем объектам по Вашей теме задания. Класс "предок" должен содержать:

  • конструктор, принимающий параметры для инициализации свойств; предусмотреть возможность инициализации свойств значениями по умолчанию; выводить в консоль строку с именем класса при выполнении конструктора
  • два свойства, которые являются общими для всех объектов; свойствам присваивать значение через конструктор;
  • один статический метод
  • один метод, который является общим для всех объектов

Создать два класса "потомка". Классы "потомки" расширить за счёт класса "предка". В обоих классах "потомка" реализовать:

  • конструктор, выводящий в консоль имя класса
  • геттер и сеттер для двух свойств класса "потомок"; выбирать свойства необходимо исходя из Вашего варианта
  • один метод, который является уникальным для данного класса

Для любого из двух класса "потомков" переопределить метод из класса "предок".

Вывести в консоль оба класса "потомка". Сравните их.
