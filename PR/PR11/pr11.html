ПР11 Использование сетевых запросов

Задача – выполнить асинхронный запрос к веб-серверу и обработать полученные данные.

Практическая работа заключается в создании асинхронного запроса веб-серверу, с передачей через адресную строку браузера логина пользователя. Сервер, получив через адресную строку логин, производит поиск в базе данных и отправляет клиенту полученный результат.

Полученный ответ от веб-сервера выводится на страницу без перезагрузки самой страницы.

На странице должны присутствовать следующие элементы

  • Текстовое поле для ввода логина, по которому производится поиск в БД
  • Кнопка с обработчиком, по нажатию на которую отправляется запрос серверу
  • Поле для вывода полученной информации

Параметры HTTP-запроса

  • Метод запроса к веб-серверу – GET
