ПР04 Динамический CSS

Подготовка

Используя VisualStudio Code создайте пустую html страницу. Установите заголовок для страницы в виде Вашей фамилии.

Подключите фреймворк Bootstrap к странице.

Разметка страницы

Добавьте на страницу тэг div, который будет служить контейнером.

Внутри контейнера создайте ещё четыре блочных элемента. Поочередно в каждый созданный элемент впишите: фамилию, имя, отчество и город проживания.

Цвет

В Bootstrap используется ряд стандартных имен классов для цветов

  • Primary
  • Secondary
  • Success
  • Danger
  • Warning
  • Info
  • Light
  • Dark
Цвета Цвет фона

Используя Bootstrap подкрасьте в разные цвета элементы, хранящие фамилию, имя, отчество и город проживания.

Использование Bootstrap

Bootstrap использует систему из 12-колонок и пяти контрольных точек. Все изменения на странице с элементами происходят благодаря использованию различных классов из состава фреймворка.

Система сеток Контрольные точки Контейнер

Созданному блоку контейнеру присвойте класс container. Используя класс row и col-{breakpoint}-{n} установите следующее поведение для элементов внутри контейнера:

  • при ширине экрана больше контрольной точки sm - блоки должны занимать по 12 колонок каждый
  • при ширине экрана больше контрольной точки md - блоки должны занимать по 6 колонок каждый
  • при ширине экрана больше контрольной точки lg - блоки должны занимать по 3 колонок каждый
  • при ширине экрана больше контрольной точки xl - блоки должны занимать по 2 колонок каждый
