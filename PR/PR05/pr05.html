ПР05 Основы JavaScript

Дано – три простейших задачи. Вам необходимо, используя язык JavaScript решить задачи согласно Вашему варианту.

Для ввода данных использовать диалоговое окно promt. А для вывода ответа на эран пользователя можно использовать либо объект console, либо диалоговое окно alert.

На html-странице вывести Ваше ФИО, номер группы и вариант задания, с описанием каждой задачи.

Номер варианта заданий соответствует номеру студента по списку в журнале +7 !!
